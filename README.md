# TALLER 1 INGENIERIA DE SOFTWARE, SISTEMA DE CHAT (FRONTEND)

Este sistema utiliza TypeScript y HTML como lenguajes y Angular como framework

Este sistema tiene las siguientes dependencias:
npm install
npm install -g @angular/cli
npm install ngx-socket-io

para iniciar el sistema (de forma local) siga los siguientes pasos:

1.- Revisar los siguientes archivos: "src/app/services/chat.ts", "src/app/services/socket.service.ts" y cambiar las URLs http://x.x.x.x:3000/ por la ip de la computadora en uso (consulte ipconfig en CMD)
2.- iniciar el codigo (ng serve) en la carpeta Frontend
3.- al terminar el codigo arrojara la siguiente URL http://localhost:4200/ donde estara alojada la web localmente
4.- debera registrarse y logearse antes de entrar a la web de chat
